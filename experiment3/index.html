Experiment 3 - Alternate Worlds

Description

This project generates a procedurally created overworld map using noise-based terrain generation and tile-based rendering. The map includes water, dirt, grass, trees, and a house. Each time the seed is changed, a new world is generated with different terrain layout and tile placement.

The world shows signs of life with animated cloud chunks that drift slowly across the screen over time. Each cloud is made up of multiple tiles to create cohesive cloud clusters. This animation brings visual movement to the overworld and adds depth to the environment.

Technical

The grid is displayed as editable ASCII characters. Users can modify the world by changing the character codes in the grid box:

  • "W" represents water
  • ":" represents dirt
  • "_" represents grass
  • "T" represents trees
  • "H" represents a house (only one is placed per map)
These codes are processed by drawGrid() and matched with specific tile positions from the tileset.?
